Notepad Stack is a simple online notepad application designed for quick note-taking and editing directly in your browser.

## Features
- Basic text editing interface for creating and managing notes.
- Client-side storage to save notes locally without needing a backend server.
- Responsive design that works on desktop and mobile devices.
- Fast, global CDN delivery with automatic HTTPS.

## Demo
Live site: [https://notepad-stack.netlify.app/](https://notepad-stack.netlify.app/)

## Tech Stack
- **Frontend**: HTML, CSS, JavaScript (likely vanilla or lightweight framework for minimal dependencies).
- **Storage**: Browser localStorage or IndexedDB for persistent note saving across sessions.

## Quick Start
1. Visit the live URL above—no installation required.
2. Start typing notes; changes auto-save locally.
3. Refresh or reopen to access saved content (data stays in your browser).

For development:
- Fork/clone the source if available on GitHub.
- Run `npm install` and `npm start` (standard setup).
- Deploy with a static hosting service.
